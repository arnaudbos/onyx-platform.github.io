Onyx

[ Get started now ] [ Sign up to our newsletter ]

Distributed Computing in Clojure

1. Write plain Clojure functions 2. Wire your program together with Clojure data 3. Test locally with core.async input tasks and trivially configured ZooKeeper testing environment 4. Alter your configuration for distributed mode and high performance messaging 5. Deploy to the cloud with Docker, Mesos, or Marathon

Fault Tolerant

Ephemeral or permanent network partition? Bare metal ripped out from underneath you? No problem, Onyx guarantees at-least-once processing with a fast, reliable technique.

Masterless

Onyx uniquely eschews a single coordinating process, making your system immune to entire classes of failures. One less dependency for dev-ops to manage.

Hybrid

Works out of the box as both a batch and stream processor with zero API changes. Process your Kafka stream and run batch jobs against your SQL database with the same cluster.

Open Source

The entire core platform and most plugins are 100% open source - and always will be. We love pull requests and suggestions.

Cloud Scale

Designed from the ground up to be transparently used on your local machine and deployed into any cloud provider with zero code changes.

High Performance

Onyx is fast, and we can prove it. Try out our repeatable, one command cloud benchmarking suite and see for yourself. Tune it to your liking.

Easy Deployment

Get your Jar file onto your nodes at your own comfort. Try SCP and S3 if you're just beginning. We recommend Docker, Mesos, and Marathon when you're ready.

Batteries Included

Instantly read and write to Datomic, SQL, Kafka, and more by leveraging our collection of open source plugins. Visualize your deployment with our Dashboard.

Ad-hoc Everything

Hook into your distributed job and collect any metrics. Emit data anywhere - to logs, to a websocket, to the dashboard. Extend and compose arbitrarily.
